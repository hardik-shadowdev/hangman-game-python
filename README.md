# Hangman Game in Python

A Python-based terminal Hangman game featuring selectable difficulty levels, bonus and boss stages, demonstrating clean game logic and core Python concepts.

## Features
- Difficulty level selection (Easy / Medium / Hard)
- Bonus stage for extra attempts
- Boss stage with increased challenge
- Input validation
- Random word selection

## Technologies Used
- Python 3

## How to Run
```bash
python Hangman2.py
